# 📚 Bachelor Thesis: The Computational Reproducibility of Marketing Research: A Specification Curve Analysis

This repository contains the code, data preparation steps, and analyses for my bachelor thesis on **the computational reproducibility of marketing research**.

The project focuses on reproducing central findings from an existing paper and extending the analysis with a **Specification Curve Analysis (SCA)** to examine how robust the reported results are across different reasonable analytical choices.

---

## 🎯 Project Goal

The main goals of this thesis are:

- 🔍 **Reproduce** a central empirical finding from the original paper
- 🧪 **Re-run the analysis** using the authors’ procedures as closely as possible
- 📈 **Test robustness** with a Specification Curve Analysis
- ⚙️ **Evaluate** how different model choices, variable operationalizations, and sample restrictions affect the result

---

## 🧠 Research Focus

This project is part of my bachelor thesis in the fields of **Marketing** and **Data Science**.

The broader research question is:

> How computationally reproducible are published findings in marketing research, and how robust are they across reasonable alternative specifications?

---

## 🛠️ Methods Used

This repository mainly uses **R** for:

- data cleaning and preparation  
- replication of regression models  
- robustness checks  
- Specification Curve Analysis  
- visualization of results  

Depending on the study and outcome variable, the analyses may include:

- Linear Regression
- Logistic Regression
- Poisson / Negative Binomial models
- Alternative model specifications
- Subsample analyses
- Control variable variations
